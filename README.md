# music

一个音乐app，因为plugin只实现了Android部分，所以iOS设备暂时无法使用

[![Packagist](https://img.shields.io/badge/download-8.3M-red.svg)](https://raw.githubusercontent.com/old-traveler/flutter_music/master/apk/app-release.apk)&nbsp;&nbsp;
[![Packagist](https://img.shields.io/badge/music-plugin-yellow.svg)](https://github.com/old-traveler/flutter_music_plugin)


## 功能介绍

- [x] 音乐播放
- [x] 音乐搜索
- [x] 音乐下载
- [x] 音乐分享
- [x] 歌手写真
- [x] 歌词展示
- [x] 歌单广场
- [x] 歌单详情
- [x] 首页分类
- [x] 直播推荐
- [x] 热门歌手
- [x] 歌手歌曲
- [x] 个人信息
- [x] 通知栏
